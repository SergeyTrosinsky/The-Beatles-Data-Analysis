# Анализ популярности: The Beatles (Data Visualization)

## Описание проекта

Проект посвящен глубокому аналитическому исследованию музыкального наследия группы The Beatles. Система позволяет интерпретировать данные о популярности треков, объемах продаж и характеристиках песен через визуализацию. Проект демонстрирует «формулу успеха» легендарного коллектива, используя современные методы `Data Science`.

Визуальная концепция проекта вдохновлена цветовой гаммой оригинальных виниловых пластинок и обложек альбомов The Beatles, что превращает сухую статистику в эстетически выверенное исследование. Проект предназначен для аналитиков данных, фанатов группы и дизайнеров, интересующихся визуализацией сложных музыкальных датасетов.

**Проект хранит и обрабатывает информацию о:**

- Продажах и чарт-листах (на основе `Best selling music artists.csv`).
    
- Характеристиках треков (на основе `The Beatles songs dataset 285x45 v0.csv`).
    
- Цветовых паттернах, адаптированных под стиль группы.
    

Проект демонстрирует навыки обработки данных с `pandas`, создания эстетичных визуализаций с `matplotlib` и `mplcyberpunk`, а также работы в среде Google Colab.

## Используемые технологии

- **Язык**: Python
    
- **Анализ данных**: Pandas, NumPy
    
- **Визуализация**: Matplotlib, mplcyberpunk
    
- **Среда**: Google Colab (Jupyter Notebook)
    
- **Дизайн**: Кастомная цветовая палитра на основе HEX-кодов альбомов.

## Как запустить проект

1. Откройте `Untitled.ipynb` в **Google Colab**.
    
2. Загрузите файлы датасетов (`.csv`) через виджет в первой ячейке.
    
3. Убедитесь, что все ячейки выполняются последовательно (Runtime -> Run all).
    
4. Результаты (графики) будут выведены непосредственно под ячейками кода.
